# DSDA Doom Batch Manager (Go)

> **Note:** This is a practice/portfolio project built for learning Go and is currently **inactive**. It is preserved here for reference and demonstration purposes.

## Description
DSDA Doom Batch Manager is a utility tool designed to help manage, configure, and run batches of demos for the `dsda-doom` source port. It reads configurations from a local `config.json` file to streamline the process of selecting IWADs, specifying demo directories, and handling outputs.

## Features
- Automated handling of `dsda-doom.exe` paths.
- Quick selection and processing of various IWADs (DOOM, DOOM2, PLUTONIA, TNT, etc.).
- Custom directory mapping for demos and outputs.

## Status
This project is currently inactive and not actively maintained. It serves as a showcase of Go programming concepts.
